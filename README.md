## 📝 简单介绍

**这是什么？**
这是一个让手机变成电脑“超级输入法”的小工具。它可以将你在手机上输入的每一段文字、每一个标点，实时同步到电脑当前光标所在的地方。

**核心亮点：**

* **语音输入神器**：电脑打字累了？打开手机语音输入法，对着手机说话，电脑屏幕上就会自动蹦出文字。
* **扫码即用**：不用连数据线，不用登录账号。只要手机和电脑在同一个 Wi-Fi，扫一下界面上的二维码就能开始。
* **聪明贴心**：你在手机上删字，电脑也会跟着退格；你在电脑上自己打字时，手机会自动感知并避让，不会把你的稿子搞乱。
* **轻量隐形**：不用的时候缩小到系统右下角的托盘，完全不占地方。

**技术实现：**
该工具采用模块化设计，由 **Python 后端 (aiohttp)** 驱动。手机端通过 **HTML5/WebSocket** 与电脑连接，利用增量同步算法确保输入几乎零延迟。同时支持 Windows、macOS 等多种系统。

### 💡 快速上手

1. **release 版本**：从 [Release 页面](https://github.com/baiyun123/InputSyncHelper/releases) 下载最新版 `InputSyncHelper.exe`。
2. **启动程序**：双击或在命令行运行 `python main.py`。
3. **扫码连接**：用手机浏览器或相机扫描窗口中的二维码。
4. **开始输入**：在手机网页输入框打字，见证文字自动“飞”进电脑。
